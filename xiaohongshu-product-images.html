📸 如何导出图片
  1. Chrome浏览器：右键点击任意图片 → "检查" → 右键图片元素 → "Capture node screenshot"
  2. Edge浏览器：按 F12 打开开发者工具 → 点击元素选择器 → 选中图片 → 右键 "捕获节点屏幕截图"
  3. 截图工具：使用系统截图（Win+Shift+S / Cmd+Shift+4）截取图片区域
  4. 批量导出：点击上方"批量导出所有图片"按钮（需联网加载html2canvas库）

💡 推荐方法：用浏览器开发者工具的 "Capture node screenshot" 功能，导出尺寸最精准
