## 产品概述

**FacePrep**（代号 Job Pilot）是一个面向中国求职者的 AI 驱动面试准备平台。

用户可以通过系统化练习、AI 个性化反馈和模拟面试，快速提升面试表现，助力成功拿到心仪 offer。

**核心场景:**
- 投递简历后，针对目标岗位进行专项备考
- 每天利用碎片时间练习高频面试题
- 模拟真实面试场景，找到薄弱环节

## 目标用户

### 主要用户群

| 用户类型 | 特征 | 核心需求 |
|---------|------|---------|
| 应届毕业生 | 面试经验少，学习意愿强 | 系统性练习 + 即时反馈 |
| 1-3年经验职场人 | 准备跳槽，时间有限 | 高效备考 + 薄弱点识别 |
| 产品/运营/设计转岗者 | 跨领域求职，缺乏针对性准备 | 岗位专题题库 + AI 指导 |
| 外企求职者 | 英文面试 + 结构化回答要求 | 标准化答题框架 |

### 用户痛点

1. **不知道怎么回答** — 缺乏结构化答题框架（STAR法则等）
2. **练习没有反馈** — 自己练自己说，不知道哪里不好
3. **题目太散** — 无法系统备考，只能靠运气
4. **临场发挥差** — 缺乏真实面试场景练习
5. **进度无感知** — 练习了多少、进步了多少看不到

---

## 核心价值主张

```
练一道题 = 知道自己哪里对了 + 哪里可以更好 + 参考答案是什么
```

**三个核心差异点:**

1. **即时 AI 反馈** — 不是对错判断，而是有深度的点评和改进建议
2. **完整内容体系** — 68道精选题 × 10大岗位类别，覆盖主流求职场景
3. **模拟面试闭环** — 从单题练习到完整面试流程，一站式备考

## 功能模块

### 1. 题库练习

**入口**: `/questions`, `/practice`

| 功能 | 说明 |
|------|------|
| 题库浏览 | 按分类/类型/难度筛选 68 道精选题 |
| 单题练习 | 文字或语音作答，提交后获取 AI 反馈 |
| 收藏题目 | 标记重点题，快速复习 |
| 练习历史 | 查看所有历史练习记录和得分 |
| 推荐题目 | 基于练习记录，智能推荐薄弱方向 |

**答题体验设计:**
- 计时器帮助用户意识到答题时间控制
- 可展开「考察点」作为提示（类似打开小抄）
- 语音输入方便移动端使用
- 提交后展示多维度 AI 评分和优化建议

### 2. AI 模拟面试

**入口**: `/interview`

| 功能 | 说明 |
|------|------|
| 创建面试 | 输入目标岗位/公司，可选上传 JD 和简历 |
| AI 出题 | 根据岗位信息生成 3-10 道定制化面试题 |
| 逐题作答 | 按顺序回答，支持跳过 |
| 面试报告 | 完成后生成综合评估报告 |

**报告内容:**
- 综合评分（0-100）
- 维度评分: 内容准确性、表达结构、语言表达、亮点挖掘
- 差距分析：与理想答案的差距
- 改进建议：具体可行的提升方向
- 优化示例：AI 生成的参考答案

### 3. 自定义 AI 题目

**入口**: `/practice/ai-custom`

| 功能 | 说明 |
|------|------|
| 输入 JD/简历 | 粘贴目标岗位描述或个人简历 |
| AI 生成题目 | 根据内容生成个性化面试题（含考察点）|
| 开始练习 | 直接进入这些题目的练习流程 |

**适用场景**: 拿到心仪岗位 JD 后，快速生成针对性练习题。

### 4. 数据仪表盘

**入口**: `/dashboard`

| 指标 | 说明 |
|------|------|
| 练习总次数 | 累计练习记录数 |
| 平均分 | 近期练习平均 AI 评分 |
| 练习连续天数 | 每日打卡连续记录 |
| 分类覆盖 | 已练习的岗位类别分布 |
| 进步趋势 | 近30天得分趋势图 |
| 薄弱分析 | 得分较低的题目类型 |

### 5. 成就系统

**入口**: `/achievements`

通过徽章激励用户持续练习：

| 成就分类 | 示例 |
|---------|------|
| 练习里程碑 | 完成第1次/第10次/第100次练习 |
| 连续打卡 | 连续练习3天/7天/30天 |
| 高分成就 | 首次获得90+分 |
| 题型完成 | 完成某类型所有题目 |
| 完美答题 | 连续3次满分 |

## 用户旅程

### 新用户首次体验

```
1. 访问首页 → 了解产品价值
        ↓
2. 可不注册，直接开始练习（匿名模式，20次上限）
        ↓
3. 选择感兴趣的题目 → 语音/文字作答
        ↓
4. 等待 AI 反馈 → 看到详细点评和建议
        ↓
5. 注册账号 → 历史记录迁移 + 解锁更多功能
        ↓
6. 继续练习或升级会员
```

### 日常用户使用路径

```
打开 APP → 查看仪表盘（看今日进度）
        ↓
选择练习方式:
  A. 题库选题 → 练习 → 查看反馈 → 收藏/继续下一题
  B. 模拟面试 → 完整流程 → 查看报告 → 针对性补练
  C. 自定义题目 → 根据目标岗位练习
```

### 求职备考场景（典型高价值用户）

```
收到面试通知
        ↓
上传 JD → AI 生成专项题目
        ↓
每天练习 3-5 道 → 追踪进步
        ↓
完成模拟面试 → 查看综合评估
        ↓
根据薄弱点补练 → 面试前二次复习
        ↓
成功拿 offer → 分享成果
```

## 会员体系

### 三档产品设计

| 等级 | 定价 | 核心权益 |
|------|------|---------|
| 免费 | 0 | 每题可练3次，总计20次（匿名用户），基础 AI 反馈（Qwen） |
| 次卡（积分）| 按需购买 | 按次消费，不限时间；深度 AI 反馈（DeepSeek）|
| 月卡 | 按月订阅 | 当月无限次；深度 AI 反馈；面试报告 |

### 试用机制

- 每个账号可一键领取 **3 积分**试用
- 领取入口: `/claim`
- 不需要绑定支付信息
- 用于体验深度 AI 反馈功能

### 付费与免费的差异

| 功能 | 免费 | 付费 |
|------|------|------|
| 基础 AI 评分 | ✓ | ✓ |
| 多维度分析 | 有限 | 完整 |
| 优化答案示例 | ✗ | ✓ |
| 教练寄语 | ✗ | ✓ |
| 差距分析 | ✗ | ✓ |
| AI 模拟面试报告 | ✗ | ✓ |
| 自定义 AI 题目 | ✗ | ✓ |

## 内容体系

### 题库分类（10类）

| 分类 | 说明 | 代表题目 |
|------|------|---------|
| FRONTEND | 前端开发 | React/Vue 原理、性能优化 |
| BACKEND | 后端开发 | 系统设计、数据库、并发 |
| PRODUCT | 产品经理 | 需求分析、用户研究、数据驱动 |
| DESIGN | UI/UX 设计 | 设计方法论、用户体验 |
| OPERATION | 运营 | 增长策略、活动运营 |
| GENERAL | 通用 | 自我介绍、职业规划、离职原因 |
| DATA | 数据分析 | SQL、统计、业务分析 |
| AI | AI 相关 | 提示词工程、AI 产品设计 |
| MARKETING | 市场营销 | 品牌、内容、渠道 |
| MANAGEMENT | 管理 | 团队管理、项目管理 |

### 题目类型（5种）

| 类型 | 英文 | 说明 |
|------|------|------|
| 自我介绍 | INTRO | 开场白、个人背景 |
| 项目经历 | PROJECT | STAR 法则叙述项目 |
| 专业技术 | TECHNICAL | 岗位专业知识 |
| 行为问题 | BEHAVIORAL | 情景/行为面试题 |
| HR 问题 | HR | 薪资、期望、职业规划 |

### 难度体系

- **1星**: 基础题，校招/初级岗位
- **2星**: 进阶题，1-3年工作经验
- **3星**: 资深题，5年以上/管理岗位

## 成就系统

激励机制基于用户行为触发，具体成就数据定义于 `src/data/achievements.ts`：

| 类型 | 触发条件 |
|------|---------|
| 首次练习 | 完成第1道题 |
| 连续打卡 | 连续N天各完成至少1道题 |
| 刷题达人 | 累计完成N道题 |
| 高分选手 | 首次获得90+分 |
| 全类别 | 覆盖所有题目分类 |
| 完美收官 | 单次模拟面试平均分90+ |

## 产品指标

### 核心健康指标（North Star）

**深度练习用户数** — 每周练习3次以上的用户数

**关联指标:**
- D1/D7/D30 留存率
- 单用户平均练习题目数
- 付费转化率（免费 → 次卡/月卡）
- 模拟面试完成率

### 参考目标值

| 指标 | 目标 |
|------|------|
| D1 留存 | > 40% |
| D7 留存 | > 20% |
| 首次练习完成率 | > 70% |
| 注册后付费转化 | > 5% |
| 月卡续费率 | > 60% |

## 竞品对比

| 功能 | FacePrep | 传统刷题 APP | 普通 AI 助手 |
|------|---------|------------|------------|
| 面试专项题库 | ✓ 68题精选 | ✓ 数量多 | ✗ |
| AI 深度反馈 | ✓ 多维度分析 | ✗ | ✓ 通用 |
| 语音输入 | ✓ | ✗ | 部分支持 |
| 模拟面试流程 | ✓ 端到端 | 部分 | ✗ |
| JD 定制题目 | ✓ | ✗ | 需要手动 |
| 进度追踪 | ✓ | ✓ | ✗ |
| 中文优化 | ✓ 全中文 | ✓ | 部分 |

**核心优势:** 将题库内容、AI 反馈质量、模拟面试流程三者整合，提供端到端的面试备考体验。
