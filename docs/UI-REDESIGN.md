# Job Pilot UI 重设计方案

> 创建时间：2026-03-01
> 状态：P0-P2 已实施，P3 待开发

---

## 一、现有问题分析

| 问题类型 | 具体表现 |
|---------|---------|
| **信息层级** | 页面元素层级不明确，用户视觉焦点分散 |
| **用户旅程** | 从登录到练习的路径较长，需要多次跳转 |
| **情感设计** | 缺乏激励元素，学习过程略显枯燥 |
| **反馈体验** | AI点评展示方式单一，缺乏可视化呈现 |
| **移动体验** | 答题页面在移动端的输入体验有提升空间 |

---

## 二、设计理念

**核心理念：「轻松备战，自信上场」**

- **轻量化**：减少认知负担，每个页面只做一件事
- **游戏化**：引入进度、成就、连续练习等激励机制
- **对话式**：AI点评以更自然的对话形式呈现
- **情感化**：用积极的视觉语言传递信心

## 五、设计系统

### 色彩体系

```
主色调（行动/品牌）
├── Primary: #6366F1 (靛蓝，现代感，专业又不失活力)
├── Primary-light: #818CF8
└── Primary-dark: #4F46E5

功能色
├── Success: #10B981 (绿，高分/做得好)
├── Warning: #F59E0B (黄，中等/待改进)
├── Error: #EF4444 (红，低分/注意)
└── Info: #3B82F6 (蓝，提示)

中性色
├── Gray-900: #18181B (主文字)
├── Gray-600: #52525B (次文字)
├── Gray-400: #A1A1AA (辅助文字)
├── Gray-100: #F4F4F5 (背景/分割)
└── White: #FFFFFF (卡片背景)
```

### 字体层级

```
H1: 28px / Bold     - 页面标题
H2: 22px / Semibold - 模块标题
H3: 18px / Semibold - 卡片标题
Body: 16px / Regular - 正文
Small: 14px / Regular - 辅助文字
Caption: 12px / Regular - 标签/说明
```

### 间距规范

```
xs: 4px   - 元素内部紧凑间距
sm: 8px   - 相关元素间距
md: 16px  - 组件内间距
lg: 24px  - 模块间距
xl: 32px  - 大区块间距
2xl: 48px - 页面留白
```

### 圆角规范

```
sm: 4px   - 小按钮、标签

lg: 12px  - 卡片
```

## 七、移动端适配要点

- **答题输入**：底部固定工具栏，键盘弹出时自动调整
- **卡片布局**：单列瀑布流，左滑收藏/右滑跳过
- **导航**：底部Tab导航（首页/练习/题库/我的）
- **手势**：支持左右滑动切换题目
